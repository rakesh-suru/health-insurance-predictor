# 🏥 Health Insurance Cost Predictor

A simple Machine Learning project that predicts individual health insurance costs based on demographic and health-related attributes using **Linear Regression**.

---

## 📌 Project Overview

This project uses a dataset of patient information to estimate medical insurance charges. It applies **data preprocessing**, **exploratory data analysis (EDA)**, and **linear regression modeling** to achieve this goal.

---

## 📊 Dataset

- **Filename**: `insurance.csv`
- **Source**: [Kaggle - Medical Cost Personal Dataset](https://www.kaggle.com/datasets/mirichoi0218/insurance)
- **Features**:
  - `age`: Age of primary beneficiary
  - `sex`: Gender
  - `bmi`: Body mass index
  - `children`: Number of dependents
  - `smoker`: Smoking status
  - `region`: Residential area
  - `charges`: Medical insurance cost

---

## 📌 Features & Steps

- ✅ Data loading and overview
- ✅ Data cleaning and encoding
- ✅ Exploratory Data Analysis (EDA)
- ✅ Correlation matrix and heatmap
- ✅ One-hot encoding for categorical variables
- ✅ Model training with `LinearRegression`
- ✅ Model evaluation (R² Score: **0.74**)
- ✅ Scatter plot: Actual vs Predicted charges
- ✅ Simple prediction demo
